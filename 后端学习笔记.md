### Node

使用 Express.js WEB 框架，添加 Typescript 支持。

```shell
mkdir myApp && cd myApp

npm init -y

npm install typescript ts-node express @types/express

npx tsc -y

mkdir src && mkdir build

vim tsconfig.josn

vim package.json
```

```json
/* tsconfig.json */
{
    "compilerOptions": {
        /*...*/
        "outDir": "./build",
        "rootDir": "./src",
    }
}
/**/
{
    /*...*/
    "scripts": {
        "dev": "npx ts-node ./src/index.ts"
    }
}
```
